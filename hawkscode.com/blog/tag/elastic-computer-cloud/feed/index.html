In this blog you will get a idea about what is amazon web services and their products. Amazon Web Services (AWS) is a cloud service platform, offering computing power,storage ,database and other service to help business scale and grow. Cloud provides IT resources on Internet with pay-per-price. You need not to buy any physical storage or server instead you can take it on rent at a very cheap price. There is a huge list of products offer by AWS but the major one is EC2, Amazon’s virtual machine service and S3, Amazon’s storage system.

Amazon Elastic Computer Cloud EC2

Amazon Elastic Computer Cloud EC2 is a web service that provide securable, scalable computer capacity in cloud. EC2 eliminates the need to invest in hardware, So you can develop and deploy applications faster. EC2 is used to run virtual machine in cloud. AWS calls these virtual machine instances.

Amazon Simple Storage Service S3

Amazon Simple Storage Service S3 provides web storage through web service interface. S3 enables customers to upload, store and download practically any file or object that is up to five terabytes (TB) in size. It is durable for objects store in service and support multiple security. To learn more about AWS cloud visit Easyshiksha.

Save costs without sacrificing performance by storing data across the S3 Storage Classes, which support different data access levels at corresponding rates. You can use S3 Storage Class Analysis to discover data that should move to a lower-cost storage class based on access patterns, and configure an S3 Lifecycle policy to execute the transfer.
